## STARLIGHT HEADLINER

![Image showing models listing architecture of the site](/docs/models.png)
### MODELS LISTING ARCHITECTURE
The models listing architecture of this site allows clients to directly enquire about the specific model on WhatsApp through the `Book a test drive` CTA, they can also view more details about models by clicking on the `View details` CTA.

---

![Image showing finance calculation system of the site](/docs//finance.png)
### FINANCE CALCULATOR
The finance calculation system of this website allows clients to calculate their monthly EMI on any specific loan amount, tenure, and interest rate, with a simple chart showing the ratio of interest and principal amount.

---

![Image showing trade-in system of the site](/docs/trade-in.png)
### TRADE-IN ENQUIRE SETUP
There's a trade-in system in this website that allows clients to exchange their old vehicle and get a new one. This form setup has four separated steps listed below;

1. VEHICLE INFORMATION
2. VEHICLE CONDITION
3. NEXT UPGRADE
4. CONTACT INFORMATION

---

![Image showing sticky control of the site](/docs/sticky.png)
### STICKY ACTION CONTROL
This site has a sticky control on the bottom right corner that allows clients to call, chat on WhatsApp, or navigate to trade-in section quickly, with an elegant open/close architecture this control plays a significant role in premium UX and conversion.
